# ctrl Brand Identity Guidelines

When building dashboards or presentation components, strictly use these hex codes to maintain the hacker/terminal aesthetic.

## Primary Colors
* **Background Deep Black:** `#0a0a0a`
* **Terminal Gray:** `#1e1e1e`
* **Text White:** `#e0e0e0`

## Accent Colors (Cyberpunk Vibe)
* **ctrl Neon Green (Success/Main UI):** `#00ff88`
* **Matrix Dark Green (Hover States):** `#00b359`
* **System Alert Red (Errors/Warnings):** `#ff4444`
* **Processing Blue (Loading States):** `#89b4fa`

## Typography
* All code blocks and terminal outputs: `Fira Code` or `Courier New`
* Standard readable text: `Inter` or `San Francisco`
